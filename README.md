# aleo-launcher

启动3个步骤：
1. 安装 docker 
```
curl -s https://get.docker.com | bash -s 
```

2. 下载配置文件
wget 
```
wget 'https://raw.githubusercontent.com/CodeBub/aleo-launcher/main/docker-compose.yaml'
```

3. 启动脚本 
```
docker compose up -d
```
